# Directives pour Copilot

- Tous les messages de commit doivent être rédigés en français.
- Lorsque des modifications affectent `histoire/Jusqu'à toucher l'écorce.md`, prends en compte l'histoire elle-même.
- Pour ce fichier, explique ce qui a changé dans le récit, le ton, les personnages, l'intrigue ou la structure narrative, et pas seulement les lignes Markdown modifiées.
- Si un changement concerne uniquement la mise en forme Markdown (titres, sauts de ligne, italique, etc.), indique-le clairement, mais cherche surtout à comprendre l'impact sur l'histoire.
- Pour les autres fichiers, utilise les règles normales de description de changements techniques.
